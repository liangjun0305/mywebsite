6. 安装php

apt-get install libxml2-dev zlib1g-dev libjpeg62-dev libpng12-dev libc-client2007b-dev

首先安装php所关联到软件。

tar zxvf httpd-2.2.14.tar.gz
./configure --with-apxs2=/usr/local/httpd/bin/apxs --with-mysql=/usr/local/mysql --with-gd --enable-gd-native-ttf --enable-ftp --with-zlib --with-openssl --with-png-dir  --with-kerberos --with-imap-ssl --with-imap --with-gettext --with-jpeg-dir --with-zlib-dir
make
make test
make install

配置php，在/usr/local/httpd/conf/httpd.conf,增加下面内容：

LoadModule php5_module modules/libphp5.so
<FilesMatch \.php$>
    SetHandler application/x-httpd-php
</FilesMatch>
