11. 安装extmail

extmail是一个基于postfix的webmail软件。里面包含了一个webmail（extmail）和一个管理程序（extman）。

首先安装几个extmail用到的软件包

apt-get install libgd-gd2-perl
perl -MCPAN -e shell
cpan>install DBI
cpan>install DBD::mysql
注意
DBD::mysql安装可能不会成功，应为他需要mysql的相关库，需要指定路径，所以可以采用源码安装
cd /root/.cpan/sources/authors/id/C/CA/CAPTTOFU/
tar zxvf DBD-mysql-4.013.tar.gz
cd DBD-mysql-4.013
perl Makefile.PL   --lib="-L/usr/local/mysql/lib/mysql  -lmysqlclient -lm -lz"　\
--cflags="-I/usr/local/mysql/include/mysql" \
--mysql_config=/usr/local/mysql/bin/mysql_config
make
make install

cpan>install Unix::Syslog

安装extmail和extman

tar zxvf extmail-1.2.tar.gz
tar zxvf extman-1.1.tar.gz
mkdir /usr/local/httpd/htdocs/extsuite
mv extmail-1.2 /usr/local/httpd/htdocs/extsuite/extmail
mv extman-1.1 /usr/local/httpd/htdocs/extsuite/extman
cd /usr/local/httpd/htdocs/extsuite/extman/docs/
