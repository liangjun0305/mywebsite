2. 安装BerkeleyDB

tar zxvf db-4.8.26.NC.tar.gz
cd db-4.8.26.NC/build_unix/
../dist/configure
make
make install
ln -s  /usr/local/BerkeleyDB.4.8/include/* /usr/include/
ln -s  /usr/local/BerkeleyDB.4.8/lib/* /usr/lib/

安装路径是：/usr/local/BerkeleyDB.4.8/。
