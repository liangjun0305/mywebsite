1. 配置postfix

1.1. 建立邮件存放的目录

mkdir /home/vmail
groupadd vmail -g 3000
useradd vmail -u 3000 -g vmail -d /home/vmail -s  /sbin/nologin
chown -R vmail:vmail /home/vmail

1.2. 建立mysql数据库

extman内置了一个postfix的mysql数据库实例。导入，改成自己的域名即可。

/usr/local/mysql/bin/mysql -u root -p < /usr/local/httpd/htdocs/extsuite/extman/docs/extmail.sql
/usr/local/mysql/bin/mysql -u root -p < /usr/local/httpd/htdocs/extsuite/extman/docs/init.sql
注意
上述extman的数据库实例中建立了几个数据库用户，如果你的数据库在网络中为还有其他的应用提供服务，那么出于安全考虑，请自行更改extmail.sql建立用户部分。

1.3. 配置postfix中mysql的选项

同样，extman将这些配置文件也提供了，直接copy过去就可以了。

cp /usr/local/httpd/htdocs/extsuite/extman/docs/mysql_virtual_* /etc/postfix/mysql/

1.4. 配置postfix中smtp通过sasl进行认证

建立/usr/local/lib/sasl2/smtpd.conf这个文件

touch /usr/local/lib/sasl2/smtpd.conf
vi /usr/local/lib/sasl2/smtpd.conf

pwcheck_method: authdaemond
log_level: 3
mech_list: PLAIN LOGIN
authdaemond_path:/usr/local/courier-authlib/var/spool/authdaemon/socket

chmod a+xr /usr/local/courier-authlib/var/spool/authdaemon

1.5. 配置postfix中smtpTLS

首先制作证书。

cd /etc/postfix/
mkdir certs
cd certs
cd certs
cp /usr/local/ssl/misc/CA.pl .

修改CA.pl文件，如下，在$REQ前面增加-nodes选项。

       } elsif (/^-newcert$/) {
            # create a certificate
            system ("$REQ -new -nodes -x509 -keyout newkey.pem -out newcert.pem $DAYS");
            $RET=$?;
            print "Certificate is in newcert.pem, private key is in newkey.pem\n"
        } elsif (/^-newreq$/) {
            # create a certificate request
            system ("$REQ -new -nodes  -keyout newkey.pem -out newreq.pem $DAYS");
            $RET=$?;
            print "Request is in newreq.pem, private key is in newkey.pem\n";

以下命令将是创建证书的过程。

./CA.pl -newca
./CA.pl -newreq
./CA.pl -sign
cat newkey.pem >> newcert.pem
rm -rf CA.pl  demoCA

myhostname = mail.test.com

myorigin = $mydomain
#mydestination = test.com, $transport_maps
local_recipient_maps = $alias_maps $virtual_mailbox_maps unix:passwd.byname

#=====================MySQL=====================
virtual_alias_maps = mysql:/etc/postfix/mysql/mysql_virtual_alias_maps.cf
virtual_mailbox_domains = mysql:/etc/postfix/mysql/mysql_virtual_domains_maps.cf
virtual_mailbox_maps = mysql:/etc/postfix/mysql/mysql_virtual_mailbox_maps.cf

virtual_mailbox_limit = 51200000
virtual_minimum_uid = 3000
virtual_transport = virtual
virtual_uid_maps = static:3000
virtual_gid_maps = static:3000

smtpd_tls_loglevel = 3
################directory setup############
readme_directory = no

html_directory = no
setgid_group = postdrop
