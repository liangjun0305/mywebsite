4. 配置Courier imapd

4.1. 制作证书。

cd /usr/local/courier-imap/share
cd /usr/local/courier-imap/share
./mkimapdcert
./mkpop3dcert

注意
注意mkimapdcert和mkpop3dcert文件里面openssl命令的路径。

imapd.pem和pop3d.pem就是IMAPS和POP3S需要的证书。

4.2. 打开IMAP服务和IMAPS服务

vi /usr/local/courier-imap/etc/imapd

IMAPDSTART=YES
MAILDIRPATH=Maildir

vi /usr/local/courier-imap/etc/imapd-ssl

IMAPDSSLSTART=YES
MAILDIRPATH=Maildir

4.3. 打开POP3服务和POP3S服务

vi /usr/local/courier-imap/etc/popd

POP3DSTART=YES
MAILDIRPATH=Maildir

vi /usr/local/courier-imap/etc/pop3d-ssl

POP3DSSLSTART=YES
MAILDIRPATH=Maildir
