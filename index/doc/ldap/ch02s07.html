7. 安装Cyrus SASL v2

此软件用于SMTP认证。

SASL全称Simple Authentication and Security Layer，是一种用来扩充C/S模式验证能力的机制。rfc地址：http://tools.ietf.org/html/rfc2222

在安装之前看清configure选项很重要。你首先要确定你的SMTP验证准备通过什么样的方式来验证。是通过数据库呢，还是系统密码，还是ldap呢?然后你希望你的SMTP提供那些验证机制，是PLAIN,还是LOGIN，还是OTP，还是等等，还是都支持。那么弄清楚之后。加上相应的configure选项。

saslauthd支持的验证机制：dce,getpwent,kerberos4,kerberos5,pam,rimap,shadow,sasldb,ldap,sia

authdaemond全称是：Courier-IMAP authdaemond，那么显而易见，支持Courier-IMAP进程验证。

pwcheck：使用一个分开的帮助进程来验证，主要为了向后兼容。

cd cyrus-sasl-2.1.23
./configure --with-openssl=/usr/local/ssl/ --enable-login  --disable-gssap --with-authdaemond
make
make install
ln -s /usr/local/lib/sasl2 /usr/lib/sasl2
