1. 从Nessus 3.x往上升级

这一节主要讲如何从Nessus 3.x升级到Nessus 3.x更高的版本。

1.1. Red Hat and SUSE

如果你使用的是Nessus RPM包安装方式，那升级将会很简单，服务器和用户的配置都会很容易的保存在新版本的配置中。

使用Nessus RPM升级方法，下载最新的Nessus RPM包从http://www.nessus.org/download/或者其他网站。

在升级之前，先停止Nessus服务的进程：

# killall nessusd
注意
killall 这个命令会停止正在进行的扫描

然后，用下面的命令升级到最新版：

# rpm -Uvh <newest release>

一旦升级完成，用下面的命令重启Nessus服务：

# /opt/nessus/sbin/nessusd -D

1.3. Solaris

下载最新的Nessus包从http://www.nessus.org/download/或者其他网站。在solaris上升级，你必须先卸载原来的版本，然后在安装新的版本。这个过程将不会删除不属于原来安装包的配置文件。

在卸载之前，先停止Nessus服务的进程：

# /etc/init.d/nessusd stop
注意
/etc/init.d/nessusd stop 这个命令会停止正在进行的扫描

要删除这个软件包，你首先要确定你删除软件包的名字，这个名字和安装包的名字是不一样的。你可以用下面的命令查看：

# pkginfo | grep nessus

下面的就是上面命令的输出：

application TNBLnessus The Nessus Network Vulnerability Scanner

在Solaris 里删除Nessus包运行下面的命令：

# pkgrm <package name>

例如，我查找到在Solaris里Nessus包的名字是TNBLnessus，然后我就用如下命令来删除它：

# pkgrm TNBLnessus

这里将不会删除不属于原来安装包的配置文件，如果原来安装包的配置文件有过改动，这些配置文件也不会被删除。

安装新的安装包运行下面命令：

# gunzip <package name>.pkg.gz# pkgadd –d ./<package name>

这里你要指定新安装包的全路径，下面是列子：

# pkgadd –d /export/home/sysadmin/Nessus-3.2.0-solaris-sparc.pkg

一旦升级完成，用下面的命令重启Nessus服务：

# /opt/nessus/sbin/nessusd -D

1.4. FreeBSD

下载最新的Nessus包从http://www.nessus.org/download/或者其他网站。在FreeBSD上升级，你必须先卸载原来的版本，然后在安装新的版本。这个过程将不会删除不属于原来安装包的配置文件。

在卸载之前，先停止Nessus服务的进程：

# killall nessusd
注意
killall 这个命令会停止正在进行的扫描

要删除这个软件包，你首先要确定你删除软件包的名字，这个名字和安装包的名字是不一样的。你可以用下面的命令查看：

# pkg_info

下面的就是上面命令的输出：

Nessus-3.0.6 A powerful security scanner

在FreeBSD 里删除Nessus包运行下面的命令：

# pkg_delete <package name>

例如，我查找到在FreeBSD里Nessus包的名字是Nessus-3.0.6，然后我就用如下命令来删除它：

# pkg_delete Nessus-3.0.6

这里将不会删除不属于原来安装包的配置文件，如果原来安装包的配置文件有过改动，这些配置文件也不会被删除。

安装新的安装包运行下面命令：

# pkg_add <newest release>

一旦升级完成，用下面的命令重启Nessus服务：

# /usr/local/nessus/sbin/nessusd -D

1.5. Enterasys Dragon IDS

下载最新关于Enterasys Dragon的Nessus包。

在卸载之前，先停止Nessus服务的进程：

# killall nessusd
注意
killall 这个命令会停止正在进行的扫描

然后，用下面的命令升级到最新版：

# installpkg <newest release>

一旦升级完成，用下面的命令重启Nessus服务：

# /opt/nessus/sbin/nessusd –D –S <IP Address>

这是一个在Dragon上升级的列子：
