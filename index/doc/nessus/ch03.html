第 3 章 支持的操作系统

Nessus是跨平台的软件，Nessus服务器软件支持的操作系统

  • Red Hat ES 3, ES 4, and ES 5 (i386 and x86-64)
  • Fedora Core 7, 8, and 9
  • SUSE 9.3 and 10.0
  • Debian 4 (i386 and amd64)
  • FreeBSD 7.0
  • Solaris 9 and 10
  • Ubuntu 7.10 and 8.04 (i386 and amd64)
  • Mac OS X 10.5
  • Windows 2000, XP, Server 2003, Server 2008, and Vista
  • Enterasys Dragon appliance running Dragon 7.2 or later

Nessus客户端软件支持的操作系统

  • Debian 4
  • Red Hat ES 4 and ES 5
  • Fedora Core 7, 8, and 9
  • SuSE 10.3
  • Ubuntu 7.10 and 8.04
  • Windows 2000, XP, Server 2003, Server 2008, and Vista
  • Max OS X 10.5
