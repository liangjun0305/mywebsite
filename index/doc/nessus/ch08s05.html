5. Enterasys Dragon IDS

下载最新的Nessus包从http://www.nessus.org/download/或者其他网站。Nessus支持的版本有：Dragon 7.2或者更高的版本。

注意
除非特别指出，不然下面的所有命令都是在root权限下运行。

用下面的命令进行安装：

# # installpkg Nessus-3.2.0-dragon-skw-i386.tgz

上面将会默认安装到/opt/nessus/，下面就算是在Dragon 7.2下的例子：

注意安装时候的输出，Dragon的所有设备都是多网卡的，所以你开始Nessus服务时候应该使用下面的命令：

# /opt/nessus/sbin/nessusd –D –S <ip address>

根据用户系统配置不一样可能会出现这样的信息提示

/sbin/ldconfig: File /usr/lib/libcprts.so is too small, not checked.

这不是错误信息，只是个消息提醒。

注意
由于Dragon IDS硬件资源的原因，在Dragon IDS安装Nessus需要一个小时的时间。
