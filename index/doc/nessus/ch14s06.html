6. 扫描报告

当扫描完成后将会产出一份扫描报告，这里面便是扫描的结果，你可以方便的将扫描结果到处成多种类型文件：html，.nbe，.nsr。

Nessus的扫描报告过滤选项更是能帮你定位漏洞，如图。

图 14.14. 扫描报告过滤


这儿有六个扫描报告过滤条件。

  • Plugin ID
  • Plugin Name
  • Port Name
  • Host Name Starts With
  • Host Name Contains
  • Report Contains
  • Plugin Severity
