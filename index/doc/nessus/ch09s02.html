2. 在线安装插件

2.1. 安装插件激活码

注意
下面命令都是假定Nessus安装目录是/opt/nessus，如果你是FreeBSd，将命令行中的/opt/nessus替换成/usr/local/nessus/就行了。

你可以去这里注册免费使用的插件激活码：http://www.nessus.org/register/，但是这个更新速度会比收费激活码慢一周。

你可以用下面命令来进行注册：

# /opt/nessus/bin/nessus-fetch -–register <license code>

下面就是安装激活码的列子：

# cat /opt/nessus/lib/nessus/plugins/plugin_feed_info.inc
PLUGIN_SET = "200901110034";
PLUGIN_FEED = "HomeFeed (Non-commercial use only)";
# /opt/nessus/bin/nessus-fetch -–register XXXX-XXXX-XXXX-XXXX-XXXX
Your activation code has been registered properly – thank you.
Now fetching the newest plugin set from plugins.nessus.org...
Your Nessus installation is now up-to-date.
If auto_update is set to 'yes' in nessusd.conf, Nessus will
update the plugins by itself.
# date
Thu Oct 13 11:15:35 EDT 2005
# cat /opt/nessus/lib/nessus/plugins/plugin_feed_info.inc
PLUGIN_SET = "200903110034";
PLUGIN_FEED = "HomeFeed (Non-commercial use only)";

2.2. 插件升级

注意
下面命令都是假定Nessus安装目录是/opt/nessus，如果你是FreeBSd，将命令行中的/opt/nessus替换成/usr/local/nessus/就行了。

用下面的命令来更新插件：

# /opt/nessus/sbin/nessus-update-plugins
