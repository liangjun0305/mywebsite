3. 测试RAC安装

通过以下命令来测试crs的状态：

root@rac-master:~# /var/oracle/product/crs/bin/crs_stat -t

Name           Type           Target    State     Host        
------------------------------------------------------------
ora....ter.gsd application    ONLINE    ONLINE    rac-master  
ora....ter.ons application    ONLINE    ONLINE    rac-master  
ora....ter.vip application    ONLINE    ONLINE    rac-master  
ora....ave.gsd application    ONLINE    ONLINE    rac-slave   
ora....ave.ons application    ONLINE    ONLINE    rac-slave   
ora....ave.vip application    ONLINE    ONLINE    rac-slave

如果得到以上输出，说明你的rac安装成功
