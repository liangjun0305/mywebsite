4. 添加虚拟共享磁盘

首先克隆另外一台服务器，克隆的时候将Reinitalize the MAC address fo all network cards选项勾选，这选项就是重新创建网卡，而不是克隆。

创建共享磁盘。

VBoxManage createhd --filename ocr.vdi --size 300 --format VDI --variant Fixed

VBoxManage createhd --filename votingdisk.vdi --size 100 --format VDI --variant Fixed

VBoxManage createhd --filename asm1.vdi --size 1024 --format VDI --variant Fixed

VBoxManage createhd --filename asm2.vdi --size 1024 --format VDI --variant Fixed

将共享磁盘挂载到rac-master上。

VBoxManage storageattach rac-master --storagectl "SATA 控制器" --port 1 --device 0 --type hdd --medium ocr.vdi --mtype shareable

VBoxManage storageattach rac-master --storagectl "SATA 控制器" --port 2 --device 0 --type hdd --medium votingdisk.vdi --mtype shareable

VBoxManage storageattach rac-master --storagectl "SATA 控制器" --port 3 --device 0 --type hdd --medium asm1.vdi --mtype shareable

VBoxManage storageattach rac-master --storagectl "SATA 控制器" --port 4 --device 0 --type hdd --medium asm2.vdi --mtype shareable

将磁盘设置成共享。

VBoxManage modifyhd votingdisk.vdi --type shareable

VBoxManage modifyhd ocr.vdi --type shareable

VBoxManage modifyhd asm1.vdi --type shareable

VBoxManage modifyhd asm2.vdi --type shareable

将共享磁盘挂载到rac-slave上。

VBoxManage storageattach rac-slave --storagectl "SATA 控制器" --port 1 --device 0 --type hdd --medium ocr.vdi --mtype shareable

VBoxManage storageattach rac-slave --storagectl "SATA 控制器" --port 2 --device 0 --type hdd --medium votingdisk.vdi --mtype shareable

VBoxManage storageattach rac-slave --storagectl "SATA 控制器" --port 3 --device 0 --type hdd --medium asm1.vdi --mtype shareable

VBoxManage storageattach rac-slave --storagectl "SATA 控制器" --port 4 --device 0 --type hdd --medium asm2.vdi --mtype shareable

启动两台服务器，启动raw内核模块。

# modprobe raw

提示
raw内核模块在debian默认是关闭的需要手动启动，并加入到开机启动中。

将内核模块加入开机启动，增加raw的内容

vi /etc/modules

raw
