1. 安装openldap

1.1. 安装BerkeleyDB

由于openldap需要用Berkeley DB的一些库文件，所以要先安装Berkeley数据库. 

tar zxvf tar/db-4.5.20.NC.tar.gz
cd db-4.5.20.NC/build_unix
../dist/configure
make
make install
ln -s  /usr/local/BerkeleyDB.4.5/include/* /usr/include/
ln -s  /usr/local/BerkeleyDB.4.5/lib/* /usr/lib/
注意
BerkeleyDB 的版本在这里我尝试了db-4.5.20,但是安装openldap时候就是test不过去,当然openldap也不能用拉.db-4.5.20.NC这个版本是没有问题的。

1.2. 安装openssl

tar zxvf tar/openssl-0.9.8e.tar.gz
cd openssl-0.9.7e
./config
make
make install
注意
openssl安装你也可以选择安装的目录。但是我觉得ssl里面的一些库文件很多的软件都需要，所以建议默认的目录。这样其他软件找ssl的库文件就很方便。不然的话你就需要把库文件链接系统的库文件目录。

1.3. 安装openldap 

cd /var/src
tar zxvf tar/openldap-stable-20080813.tgz
cd openldap-2.3.38
env CPPFLAGS="-I/usr/local/BerkeleyDB.4.5/include" LDFLAGS="-L/usr/local/BerkeleyDB.4.5/lib"
./configure --prefix=/usr/local/openldap/
make depend
make
make test
make install
注意
在make test的时候请多注意，有可能会出现错误信息的。这与你前面安装的软件版本有关。没有错误报告，那恭喜，你openldap编译成功！
